◼️ 個体の性質から生まれる個体群の性質

原理

$$ N_{t+1} = N_{t} + b N_{t} - d N_{t} $$

$$ N_{ t+1 }=(s+b)N_{ t }={ \lambda }{ N_{ t } }\tag{6} $$

個体数の時間変化 $$ \begin{equation} N_{t} = {\lambda}{N_{t}}\tag{漸化式} \end{equation}\[10pt]　 $$$$ \begin{equation} N_{t} = {\lambda}^{t}{N_{0}}\tag{解} \end{equation} $$

レズリー行列

$$ \mathbf{L}=\left( \begin{array}{ccccc}{s_{0} b_{1}} & {s_{1} b_{2}} & {\cdots} & {s_{m-1} b_{m}} & {0} \ {s_{0}} & {0} & {0} & {\cdots} & {0} \ {0} & {s_{1}} & {0} & {\cdots} & {0} \ {0} & {0} & {\ddots} & {0} & {0} \ {0} & {0} & {0} & {s_{m-1}} & {0}\end{array}\right), ; \mathbf{n}(t)=\left( \begin{array}{c}{n_{0}(t)} \ {n_{1}(t)} \ {n_{2}(t)} \ {\vdots} \ {n_{m}(t)}\end{array}\right) $$

個体数の時間変化 $$ \begin{align} \left( \begin{array}{c} { n_{ 0 }(t+1) } \ { n_{ 1 }(t+1) }\ { n_{ 2 }(t+1) } \ { \vdots } \ { n_{ m }(t+1) } \end{array} \right) = \left( \begin{array}{ccccc} { s_{ 0 }b_{ 1 } } & { s_{ 1 }b_{ 2 } } & { \cdots } & { s_{ m-1 }b_{ m } } & { 0 } \ { s_{ 0 } } & { 0 } & { 0 } & { \cdots } & { 0 } \ { 0 } & { s_{ 1 } } & { 0 } & { \cdots } & { 0 } \ { 0 } & { 0 } & { \ddots } & { 0 } & { 0 } \ { 0 } & { 0 } & { 0 } & { s_{ m-1 } } & { 0 } \end{array} \right) \left( \begin{array}{c} { n_{ 0 }(t) } \ { n_{ 1 }(t) }\ { n_{ 2 }(t) } \ { \vdots } \ { n_{ m }(t) } \end{array} \right) \end{align} $$ $$ \begin{equation} {\bf n}(t+1) = {\bf L },{\bf n}(t)\tag{漸化式} \end{equation}\[10pt] $$$$ \begin{equation} {\bf n}(t) = {\bf L }^{ t },{\bf n}(0)\tag{解} \end{equation} $$

${\bf n}{(t)}$ の一般解は、$\bf L$ の固有値($\lambda_{1} \ge \lambda_{2} \ge \cdots$)と固有ベクトル(${\bf u}_{1} \ge {\bf u}_{2} \ge \cdots$)によって $$ {\bf n}(t) = {g_1}{\bf \lambda}_{1}^{t}{\bf u}_{1} +{g_2}{\bf \lambda}_{2}^{t} {\bf u}_{2} + \cdots　\tag{一般解} $$ と表され、任意の初期状態 ${\bf n}{(0)}$ で、十分な時間 $t$ に対しては、 $$ {\mathbf n}(t)\propto {\mathbf \lambda}^{t}{\bf u}_{1}, \hspace{10mm} ({\lambda}={\lambda}_{1} ) \tag{8} $$

と見ることができる。${\bf u}{1}$の要素は以下の比に保たれる $$ {\mathbf u}{1} = \left( \begin{array}{c} { u_{ 0 }(t) } \ { u_{ 1 }(t) }\ { u_{ 2 }(t) } \ { \vdots } \ { u_{ m }(t) } \end{array} \right) \propto \left( \begin{array}{c} { l_{ 0 }{{\lambda}^{-0}} } \ { l_{ 1 }{{\lambda}^{-1}}}\ { l_{ 2 }{{\lambda}^{-2}}} \ { \vdots } \ { l_{ m }{{\lambda}^{-m}} } \end{array} \right).\[10pt]\tag{9} $$

  • 個体数は $t$ についれ $\lambda$ 倍ずつ変化する指数関数に従う。
  • 個体群内の齢構成は ${\bf u}_1$ に従う一定の比率に保たれる(安定齢分布)。
